# Avocado_price_prediction_-_Region_Analysis
Machine learning project to predict avocado prices and classify organic vs conventional avocados using real retail data.

Performed data cleaning, EDA, and feature engineering

Built regression and classification models

Random Forest gave the best performance

Extracted useful business insights on pricing and regions

Models Used

Linear Regression, Gradient Boosting, Random Forest

Logistic Regression, Decision Tree, Random Forest Classifier

Tech Stack

Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn
